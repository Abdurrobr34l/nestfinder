# NestFinder 🏠

A modern real estate listing platform built for the Bangladesh property market.
Users can search properties by city, browse listings, view full property details,
and contact owners directly via phone or WhatsApp.

## Features

- Search properties by city (Dhaka, Chittagong, Sylhet, Rajshahi)
- Filter by type, price range, bedrooms
- Property details page with owner contact
- Blurred phone number — revealed only after login
- WhatsApp direct contact button
- Submit your property form for owners
- Role-based dashboard (User + Admin)
- Admin can manage listings and users
- Charts and analytics in admin dashboard
- AI chatbot — describe what you need, get suggestions
- AI review summary on property details page
- Light and dark mode
- Fully responsive (mobile, tablet, desktop)
